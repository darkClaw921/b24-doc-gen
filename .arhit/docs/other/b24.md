# b24

Frontend Bitrix24 SDK bootstrap (lib/b24.ts). initB24() lazy-initializes B24Frame via initializeB24Frame() with module-scope caching. getB24Auth() returns {accessToken, refreshToken, expiresAt, domain, memberId, scope}. getCurrentDealId() reads deal id from placement options. getCurrentPlacement() returns placement code. getB24AuthHeaders() builds X-B24-* headers for backend calls. Gracefully handles non-iframe context: isB24Available() lets components render fallback UI.
