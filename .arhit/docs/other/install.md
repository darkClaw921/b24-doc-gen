# install

Backend install routes. GET /api/install/status returns {installed, adminUserIds, dealFieldBinding, portalDomain, installedAt}. POST /api/install upserts AppSettings (id=1) with sanitized adminUserIds, portalDomain from request.b24Auth, optional dealFieldBinding. POST /api/install/register-placements: (a) reads handler URL from body.handlerUrl > FRONTEND_PUBLIC_URL > PUBLIC_URL > FRONTEND_URL and rejects non-HTTPS to avoid Bitrix24's ERROR_WRONG_HANDLER_URL; (b) calls placement.list as a diagnostic and returns availablePlacements; (c) calls placement.bind ONLY for CRM_DEAL_DETAIL_TAB (the app entry / DEFAULT placement is set via the manifest's handler URL, not via API — binding it returns ERROR_ARGUMENT); (d) returns {results, availablePlacements, placementListError}. Logs every bind attempt with code and params. Helper toAppSettings(row) converts Prisma row to shared AppSettings shape.
