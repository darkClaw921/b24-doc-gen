# formulaEngine

Sandboxed mathjs-движок формул шаблонов. math=create(all) с заблокированными небезопасными примитивами (import/createUnit/simplify/derivative/...). Helper-функции (импортируются в math instance): if(cond,a,b), concat(...args), format(value,pattern) (0/0.00/money/usd/eur/проценты), dateFormat(date,fmt) (iso/date/datetime + токены yyyy/yy/MM/dd/HH/mm/ss), today(fmt?) — текущая дата на момент генерации, форматируется тем же formatDate, по умолчанию dd.MM.yyyy, не зависит от контекста сущностей, добавлена в KNOWN_NON_ENTITY_SYMBOLS, upper(s), lower(s). Product-хелперы как замыкания в scope evaluateExpression (productCount/productSum/productGet/productImage). ENTITY_SYMBOLS=['DEAL','CONTACT','COMPANY','ASSIGNED']. evaluateExpression оборачивает сущности в Proxy withMissingFieldDefaults (отсутствующее/null поле → ''). Публичный API: validateExpression, evaluateExpression, extractDependencies. Покрыто formulaEngine.test.ts (включая тесты today()).
