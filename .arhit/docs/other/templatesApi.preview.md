# templatesApi.preview

Frontend API client method (apps/frontend/src/lib/api.ts, inside templatesApi). Signature: preview(id: string, dealId: number, fieldValues?: Record<string,string>, signal?: AbortSignal) => Promise<TemplatePreviewResponseDTO>. Issues POST /templates/:id/preview with JSON body { dealId, fieldValues } (typed via TemplatePreviewRequestDTO). Returns TemplatePreviewResponseDTO { docxBase64, tags, formulas, fields } (legacy { html } field removed). Supports AbortSignal so a debounced re-fetch (triggered when manual field values change on GeneratePage) can cancel an in-flight request. Errors surface as ApiError via the shared apiRequest helper (handles both wrapped and flat backend error envelopes); a 400 is returned when the template has no originalDocx.
