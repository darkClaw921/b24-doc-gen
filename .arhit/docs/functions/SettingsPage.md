# SettingsPage

Admin-only settings page (apps/frontend/src/pages/SettingsPage.tsx). Sections: (1) Deal file-field binding — select/create UF_CRM_* file field for generated docx (settingsApi.dealFields/createField/update). (2) App admins picker — debounced user search (usersApi.search) + selection map, saved via settingsApi.update {adminUserIds}. (3) Места встройки (placements) — lists bound handlers (placementsApi.list) each with a Trash2 unbind button gated behind a confirm dialog (placementsApi.unbind); PLUS an 'Добавить место встройки' picker that fetches the catalog (placementsApi.catalog), shows a <select> of CRM deal embedding locations (flags already-bound entries with 'уже встроено' via boundPlacementCodes Set), defaults to first not-yet-bound location, and a 'Встроить' button calls bindMutation (placementsApi.bind) to embed/re-embed. saveMessage banner reports ok/error for all mutations. Save bar persists field-binding + admins. Create-field dialog wraps crm.deal.userfield.add.
