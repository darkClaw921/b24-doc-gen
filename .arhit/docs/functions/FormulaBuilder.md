# FormulaBuilder

Modal dialog for building/editing template formulas. Two-column grid: left column holds label, tagKey (auto-slug via generateTagKey with tagKeyDirty flag to respect manual edits), monospaced textarea with caret-aware token insertion, operator palette (+ - * / , == != > < >= <=), helper palette (if/concat/format/dateFormat/upper/lower) — обе палитры теперь оборачивают каждую кнопку в RichTooltip с богатым содержимым: для функций — HelperTooltipContent (signature, описание, таблица аргументов, пример), для операторов — символ + одностроковое описание из OPERATOR_DOCS. Метаданные функций берутся из общего справочника HELPER_DOCS (lib/formulaHelp.ts), локально хранится только snippet вставки и cursorOffset. Опциональный preview-панель (если задан testDealId), live валидация (validateLocally sync + debounced 500ms validateRemote). Insert button blocked until valid + label + tagKey + expression filled. On submit calls onInsert({tagKey, label, expression, dependsOn}). Экспортирует HelperTooltipContent для повторного использования в EditorFormulaTooltip.
