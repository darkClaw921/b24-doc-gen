# FormulaBuilder

Модалка конструктора формул (shadcn Dialog). Слева — поле «Название формулы», авто-tagKey (generateTagKey + флаг tagKeyDirty), монопространственная textarea выражения, палитры операторов и helper-функций (if/concat/format/dateFormat/upper/lower) с RichTooltip-подсказками из HELPER_DOCS/OPERATOR_DOCS (lib/formulaHelp.ts), отдельная палитра «Товары» (productSum/productCount/productGet/productImage), блок предпросмотра (при testDealId → formulasApi.evaluate). Справа — палитра быстрого доступа «Часто используемые формулы» (история ранее вставленных формул из lib/quickFormulas.ts; пагинация по 6 чипов со стрелками; чип подписан именем формулы, тултип показывает имя+выражение; handleQuickFormula вставляет выражение в каретку, заполняет пустое «Название» именем формулы и поднимает формулу наверх истории), полный FieldPicker (handleFieldSelect просто вставляет токен поля) и блок зависимостей. Live-валидация: validateLocally синхронно + debounce 500мс validateRemote. handleSubmit записывает формулу в историю через recordRecentFormula и вызывает onInsert({tagKey,label,expression,dependsOn}). Экспортирует HelperTooltipContent для EditorFormulaTooltip. Прежний блок «Часто используемые поля» (quickFields.ts) заменён на «Часто используемые формулы».
