# parseBulkOptions

Хелпер в ManualFieldBuilder.tsx. Разбирает вставленный блок текста в массив вариантов select-поля: по одной непустой строке на вариант. Параметр splitValue: при true (режим mapped) каждая строка делится на label и value по первому табу или прогону из 2+ пробелов (часть до — название для пользователя, часть после — подставляемое значение); строка без разделителя → вариант только с label. При false (режим direct) вся строка целиком становится вариантом (без разбивки, чтобы значения с двойными пробелами не ломались). Используется кнопкой 'Разобрать' (addBulkOptions) — добавляет распознанные варианты к существующим с дедупликацией по label и удалением пустых placeholder-строк.
