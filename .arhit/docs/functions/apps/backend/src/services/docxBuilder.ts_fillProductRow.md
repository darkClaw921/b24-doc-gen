# apps/backend/src/services/docxBuilder.ts:fillProductRow

Функция fillProductRow(rowHtml, product, index) подставляет значения конкретного товара в шаблонную строку таблицы. Заменяет data-product-field на значение поля ProductRow, data-product-image на img tag с IMAGE_BASE64, data-product-index на порядковый номер (1-based).
