# ManualFieldBuilder

Модальный диалог настройки поля для ручного заполнения (admin). Поддерживает типы text|textarea|number|date|select. Для type='select' показывает секцию: переключатель режима подстановки (valueMode 'direct' — подставлять выбранное / 'mapped' — сопоставить со значением) и редактируемый список вариантов options[{label,value}] (в режиме mapped редактируются оба поля, в direct — только label). Блок 'Значение по умолчанию' для select — выпадающий список меток вариантов (defaultValue = метка предвыбранного варианта). handleSubmit для select требует ≥1 вариант с непустым label и чистит options (trim, без пустых). Результат ManualFieldBuilderResult расширен полями options и valueMode.
