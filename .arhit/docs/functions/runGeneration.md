# runGeneration

Framework-agnostic ядро генерации .docx. Шаги: загрузка template+formulas+fields+AppSettings → detectProductUsage → deal context → evaluate formulas → resolveManualFieldValues → buildDocxFromTemplate → disk upload → опциональная привязка к UF_CRM_* через crm.item.update (одиночное поле: [fileName,base64] заменяет; множественное: существующие {id} из resolveExistingFileRefs + новый [fileName,base64] = добавление без потери прежних) → опциональный crm.timeline.comment.add. Таймлайн гейтится effectiveAddToTimeline = globalAddToTimeline(AppSettings.addToTimeline) && theme.addToTimeline: глобальный выключатель в настройках имеет приоритет — при выключении комментарий не пишется ни для одной темы. Возвращает GenerationResult {fileId, downloadUrl, fileName, formulas, binding, timeline, warnings[]}. Детерминированные ошибки — GenerationError с kind.
