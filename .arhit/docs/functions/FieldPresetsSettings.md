# FieldPresetsSettings

Секция настроек 'Списочные поля (готовые списки)' (frontend). CRUD пресетов выпадающих списков через fieldPresetsApi, своя TanStack Query ['field-presets'] + мутации create/update/delete (инвалидируют тот же ключ — синхронно с пикером в ManualFieldBuilder). Список пресетов с изменить/удалить, диалог создания/редактирования с input названия и общим SelectOptionsEditor. Рендерится как section в SettingsPage.
