# SelectOption

Интерфейс в packages/shared/src/types.ts: { label, value } — один вариант select-поля. label показывается в выпадающем списке и является идентичностью варианта (что хранит/шлёт фронт). value — 'реальное' значение, подставляемое в документ при valueMode='mapped'. Связанные типы: SelectValueMode ('direct'|'mapped'), TemplateFieldType (добавлен 'select').
