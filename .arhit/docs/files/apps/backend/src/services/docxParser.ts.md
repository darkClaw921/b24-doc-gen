# apps/backend/src/services/docxParser.ts

@deprecated LEGACY-модель. parseDocxToHtml(buffer) конвертирует .docx в HTML через mammoth.js (кастомный defaultStyleMap; возвращает {html, messages}; бросает DocxParseError). Рендеринг/preview больше НЕ опираются на этот HTML — форматированием владеет оригинальный .docx (buildDocxFromTemplate). parseDocxToHtml вызывается только один раз при загрузке шаблона (templates.ts POST /upload) для заполнения колонки Template.contentHtml, по которой generationPipeline.detectProductUsage эвристически решает, грузить ли товары/картинки. Переходный helper, кандидат на замену плейсхолдер-детектором и удаление.
