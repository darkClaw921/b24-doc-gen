# apps/backend/src/services/portalAuth.ts

Portal OAuth helper. savePortalTokens() шифрует accessToken/refreshToken/applicationToken через tokenCrypto.encryptToken перед upsert в AppSettings (id=1). getFreshPortalAuth() читает строку, проверяет expiry (skew 60с) и возвращает расшифрованный accessToken через decryptToken; если токен протух — вызывает refreshPortalTokens. refreshPortalTokens() расшифровывает refreshToken, шлёт его в https://oauth.bitrix.info/oauth/token/ (grant_type=refresh_token, B24_CLIENT_ID/B24_CLIENT_SECRET из env), парсит ответ, сохраняет новую пару через savePortalTokens (которая заново зашифрует). PortalAuthError с kind: not_installed|missing_tokens|missing_client_credentials|refresh_failed. Decryption tolerant к legacy plaintext (см. tokenCrypto) — миграция бесшовная.
