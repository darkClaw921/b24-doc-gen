# apps/backend/src/routes/generate.ts

POST /api/generate — produce a .docx from a saved template, upload to disk, optionally bind to deal and post timeline comment. Pipeline: 1) load template+formulas+theme; 2) compute effectiveFieldBinding = template.theme.dealFieldBinding ?? settings.dealFieldBinding ?? null; 3) effectiveAddToTimeline = template.theme.addToTimeline; 4) getDealContext for formula evaluation; 5) evaluate formulas; 6) buildDocxFromHtml; 7) disk.storage.getforapp + disk.folder.uploadfile; 8) if effectiveFieldBinding set: query getDealFields() to detect isMultiple, then either set scalar or append fileId to existing array (read from context.DEAL[fieldName]); fallback to value-shape detection if getDealFields fails; 9) if effectiveAddToTimeline true: crm.timeline.comment.add with FILES=[[fileName, base64]] so the .docx attaches as a real file in the timeline. Returns {fileId, fileName, downloadUrl, formulas, binding, timeline, warnings}.
