# apps/frontend/src/App.tsx

Корневой компонент фронтенда. Объявляет маршруты React Router: /install (InstallPage), /templates (TemplatesPage), /templates/:id/edit (TemplateEditorPage), /generate (GeneratePage), /settings (SettingsPage). Корень / и 404 редиректят на /templates. Все страницы-заглушки на Фазе 1, реализация начиная с Фазы 2.
