# apps/frontend/src/pages/TemplateEditorPage.tsx

Admin WYSIWYG editor at /templates/:id/edit. Loads template via templatesApi.get and themes via themesApi.list (TanStack Query). Local state: editor (TipTap), contentHtml, name, themeId, dirty, saveError, savedAt, formulasByKey (Record<tagKey, FormulaInputDTO>), builderOpen, editingFormulaKey. Click-to-edit: a delegated click listener installed on editor.view.dom catches clicks on span[data-formula-key] pills, sets editingFormulaKey and opens FormulaBuilder pre-filled via initialValues. handleInsertFormula has two modes: EDIT — locates the formulaTag node by old tagKey via doc.descendants, calls tr.setNodeMarkup with new attrs, updates formulasByKey (preserving id, dropping old key if renamed); INSERT — uses editor.chain().insertFormulaTag and merges into formulasByKey. Save serializes formulasByKey via PUT /api/templates/:id. Содержит вложенный компонент EditorFormulaTooltip — Google Sheets-style hover-подсказка для пилюль формул: слушает делегированные mouseover/mouseout на editor.view.dom, через 200ms показывает portal-окно с label + expression + chips зависимостей (DEAL/CONTACT/COMPANY с цветовой маркировкой) + автоматически собранным списком используемых функций через extractUsedHelpers + HelperTooltipContent. pointer-events: none, чтобы не мешать click-to-edit.
