# apps/frontend/src/pages/TemplatesPage.tsx

Главная страница админки шаблонов. Шапка содержит ThemeSidebar слева и main с заголовком + поиск + список шаблонов. Для админов в шапке доступны кнопки 'Настройки' (navigate('/settings')) и 'Загрузить шаблон' (Dialog с TemplateUploader), обёрнутые в AdminOnly. Темы выбираются через ThemeSidebar, выбранная тема хранится локально, при выборе подгружается templatesApi.list({themeId,search}) через TanStack Query, search debounced 250ms. Карточки шаблонов отображают имя/тему/количество формул, для админов кнопки 'Webhook' (открывает WebhookManagerDialog) и 'Открыть' (navigate('/templates/:id/edit')). useCurrentRole используется для условного рендера админ-действий.
