# apps/frontend/src/pages/TemplatesPage.tsx

Главная страница админа Phase 3. Layout: ThemeSidebar слева + main справа с header (название, кнопка 'Загрузить шаблон' открывает Dialog с TemplateUploader, disabled пока тема не выбрана), search-input (debounced 250ms через useDebouncedValue) и сеткой карточек шаблонов. Состояние: selectedThemeId, searchInput, uploadOpen. Запросы через templatesApi.list({themeId, search}) с queryKey: ['templates', {themeId, search}]. Карточка показывает имя, имя темы, formulasCount, флаг hasOriginalDocx и кнопку 'Открыть' (navigate /templates/:id/edit). После успешной загрузки нового шаблона — refetch + navigate в редактор.
