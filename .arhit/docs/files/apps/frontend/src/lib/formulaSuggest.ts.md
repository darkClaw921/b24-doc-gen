# apps/frontend/src/lib/formulaSuggest.ts

«Обучение по примеру» для автоподсказки формул в редакторе шаблона (TemplateEditorPage). Запоминает пары «шаблон строки → формула»: при вставке формулы вместо выделения сохраняется шаблон ВСЕЙ строки (абзаца) с маркером SELECTION_MARK (⟦⟧) на месте выделения и маскированием чисел, в localStorage (FORMULA_MEMORY_KEY, кап FORMULA_MEMORY_MAX=200). Тип FormulaMemoryEntry {pattern,label,expression,dependsOn}. normalizeLine(text) — lowercase + схлоп пробелов + числа→#. buildLinePattern(lineText,startOffset,endOffset) — вставка маркера на место выделения + нормализация (null для пустой строки). dice(a,b) — коэффициент Сёренсена-Дайса по биграммам. findFormulaSuggestion(pattern,list) — точное совпадение шаблона, иначе лучший нечёткий матч ≥ SUGGEST_THRESHOLD=0.8. addFormulaMemory(list,entry) — дедуп по pattern, в начало, кап. loadFormulaMemory/saveFormulaMemory — localStorage с защитой. Свободен от React/DOM для unit-тестов (formulaSuggest.test.ts). Интеграция в TemplateEditorPage: captureLinePattern/applySuggestion/слушатель выделения/портал-всплывашка suggestion.
