# apps/frontend/src/lib/quickFormulas.ts

Чистые хелперы и константы палитры «Часто используемые формулы» в FormulaBuilder. Заменяет прежний quickFields.ts (история полей). Тип QuickFormula {label, expression}. Константы: QUICK_FORMULAS_PAGE_SIZE=6, QUICK_FORMULAS_MAX=20, QUICK_FORMULAS_STORAGE_KEY='b24dg:recent-formulas'. DEFAULT_QUICK_FORMULAS — сид типовых выражений (ФИО контакта, сумма деньгами, дата сделки, сумма товаров, НДС 20%, ответственный). loadRecentFormulas()/saveRecentFormulas(list) — чтение/запись истории вставленных формул в localStorage, терпят повреждённый JSON и недоступный storage, отфильтровывают записи с пустым expression. mergeQuickFormulas(recent) — слияние истории с дефолтами: недавние первыми, дедуп по expression, кап QUICK_FORMULAS_MAX. addRecentFormula(list, expression, label) — чистый редьюсер: переносит выражение в начало, дедуп по expression, кап. Свободен от React-импортов для unit-тестов (quickFormulas.test.ts, node:test через tsx). Бизнес-логика: большинство документов используют одни и те же выражения, поэтому ранее вставленные формулы предлагаются для переиспользования одним кликом.
