# packages/shared/src/types.ts

Пакет общих TypeScript-типов @b24-doc-gen/shared. Используется и на фронтенде, и на бэкенде. Содержит: AppSettings (singleton-настройки с id=1, adminUserIds как number[], dealFieldBinding), Theme (группа шаблонов), Template (шаблон с contentHtml + formulas + originalDocxBase64), Formula (tagKey, label, expression, dependsOn: FormulaDependencies), FormulaDependencies (deal/contact/company: string[]), DealField (дескриптор поля Bitrix24 с code/title/type/items), EntityValues (Record<string, unknown>), FormulaContext (DEAL/CONTACT/COMPANY: EntityValues), FormulaEvaluationResult (результат вычисления формулы с value/rawValue/error), B24AuthPayload (userId/domain/isAppAdmin/tokens), InstallRequest/Response, TemplatePreviewResponse, GenerateRequest/Response, ApiError, AppRole ('admin' | 'user'). Только типы, без runtime-кода.
