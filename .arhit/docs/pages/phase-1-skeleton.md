Фаза 1 — Каркас проекта b24-doc-gen. Создана структура pnpm monorepo с тремя пакетами: apps/frontend (Vite + React 18 + TypeScript + Tailwind CSS + shadcn/ui), apps/backend (Fastify + TypeScript + Prisma + SQLite), packages/shared (общие TS-типы). Backend имеет GET /health endpoint на порту 3001, frontend стартует на порту 5173 с проксированием /api на backend. Prisma схема содержит модели AppSettings, Theme, Template, Formula, миграция init применена (dev.db). Все три пакета проходят tsc --noEmit. Подробная документация в architecture.md.
