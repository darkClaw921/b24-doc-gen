Руководство администратора по разметке .docx-шаблонов плейсхолдерами для генератора (docxTemplateEngine/buildDocxFromTemplate). Файл: docs/template-markup-guide.md. Содержит: (1) синтаксис одинарных фигурных скобок { } — формулы {tagKey}, ручные поля {fieldKey}, товарный цикл {#products}...{/products} с зарезервированными полями (INDEX, PRODUCT_NAME, PRICE, QUANTITY, MEASURE_NAME, DISCOUNT_SUM, TAX_RATE, SUM, SORT, PRODUCT_ID, ID) и картинки {%imageTag} / {%PREVIEW_PICTURE_BASE64} / {%DETAIL_PICTURE_BASE64}; (2) привязку формул/полей в UI-редакторе через панель Теги шаблона (статусы formula/field/reserved/unbound), FormulaBuilder и ManualFieldBuilder; (3) пошаговую миграцию 12 шаблонов из папки 'шаблоны для первичного обращения' — замена человекочитаемых заглушек (автомат из КК / ХХХ / тянется из справочника / _____) на теги с сохранением форматирования 1:1; (4) чек-лист приёмки (нет unbound, preview=Word, .docx 1:1). Источник истины по синтаксису — шапка docxTemplateEngine.ts; зарезервированные теги — templateTags.ts RESERVED_PRODUCT_TAGS.
